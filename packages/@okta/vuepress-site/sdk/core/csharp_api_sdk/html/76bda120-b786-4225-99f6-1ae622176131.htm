Installation

The SDK is distributed as a Portable Client Library with support for .NET 4, Windows 8, Windows Phone 8.1, and Silverlight 5. If you're targeting one of those platforms, read on.

Installation (NuGet)

  1. Right-click your project or solution in the Solution Explorer

  2. Click Manage NuGet Packages

  3. Search for Okta

  4. Install

Build

  1. Ensure you have NuGet and Visual Studio 2012 Professional or higher

  2. Clone our GitHub repo

  3. Double-click build.bat

  4. You should now have Okta.Core.dll in ./bin/Release
