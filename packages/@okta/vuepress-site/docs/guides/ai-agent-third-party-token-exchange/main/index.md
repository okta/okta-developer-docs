Learn how to configure token exchange for third-party AI agents so that you can securely request and use credentials to access protected resources on behalf of authenticated users.

> **Note**: To enable AI agent token exchange, you must first subscribe to Okta for AI Agents. See your Okta account team to enable the feature.

---

#### Learning outcomes

- Understand how to set up the token exchange flow for a third-party AI agent.

#### What you need

- An Okta org that's subscribed to Okta for AI Agents.
- An Okta user account with the super admin role.
