


**Note**: You will need the entitlement bundleID to import an existing entitlement object into Terraform. You can retrieve this ID in either the Okta Admin Console or using [Entitlement Bundles API](https://developer.okta.com/docs/api/iga/openapi/governance.api/tag/Entitlement-Bundles/).


```shell
terraform import okta_entitlement_bundle.example <entitlement_bundle_id>
```
