###  Supported capabilities

This guide covers submissions that use API Integration Actions for the following capabilities:

* Universal Logout (must also have SSO capability)
* Provisioning
* Provisioning with entitlement management

> **Notes:**
> * Universal Logout integrations are only supported for the SAML 2.0 and OIDC protocols. If you want to submit a Universal Logout integration with provisioning, you must also submit an SSO integration with either SAML 2.0 or OIDC.
> * Entitlement management is supported for provisioning integrations that manage entitlements.
> * There are protocol-specific limitations on integrations in the OIN. See [OIN limitations](/docs/guides/submit-app-prereq/main/#oin-limitations).
