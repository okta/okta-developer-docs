#### OIN catalog properties

1. In the **OIN catalog properties** section, enter the following information:

    | <div style="width:150px">Property</div>| Description  |
    | ----------------- | ------------ |
    | **Display name** `*` | The name of your integration, as it should appear in the OIN.<br>The maximum field length is 64 characters. |
    | **Description** `*` | A description of your app and the benefits it provides to customers. See [App description guidelines](/docs/guides/submit-app-prereq/main/#app-description-guidelines). |
    | **Logo** `*` | A PNG, JPG, or GIF file of a logo to accompany your integration in the catalog. The logo file must be less than one MB. See [Logo guidelines](/docs/guides/submit-app-prereq/main/#logo-guidelines). |

    `*` Required properties

1.  Click **Configure your integration**.
